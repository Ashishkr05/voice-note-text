# Voice Notes Transcription App

A modern web application that records voice notes and transcribes them to text using OpenAI's Whisper API. Built with Next.js, Express, and Tailwind CSS.


## Features

- 🎤 Voice recording with real-time feedback
- 📝 Automatic speech-to-text transcription
- 📱 Fully responsive design
- 🔒 Secure file handling

## Tech Stack

- **Frontend:**
  - Next.js 14
  - Tailwind CSS
  - Lucide Icons
  - React Hooks

- **Backend:**
  - Express.js
  - OpenAI Whisper API
  - Multer
  - CORS

## Quick Start

### Prerequisites

- Node.js 18+
- OpenAI API key
- npm or yarn

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm start
```
